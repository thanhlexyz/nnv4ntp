\section{Conclusion}

\begin{frame}{Concluding remarks}
    \begin{itemize}
        \item \textbf{Contribution:} Formal verification framework for \gls{dl}-based mobile traffic prediction $\to$ provable robustness against adversarial traffic injection.
        \item \textbf{Approach:} Input perturbations (hyperrectangle) + output properties (over/underprovisioning bounds) $\to$ \texttt{NeuralSAT} for sound guarantees.
        \item \textbf{Finding:} Asymmetric robustness on Telecom Italia Milan: resilient to overestimation under moderate perturbations; highly vulnerable to underestimation attacks.
        \item \textbf{Future work:} Broader evaluation (more datasets and models); verifier in the adversarial retraining loop.
    \end{itemize}
\end{frame}
